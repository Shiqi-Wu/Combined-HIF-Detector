# `src/koopman`

Koopman-based HIF localization methods.

- `phi_train.py`: standard parametric Koopman training
- `inverse_eval.py`: standard Koopman inverse evaluation
- `residual_train.py`: residual Koopman training
- `residual_eval.py`: residual Koopman evaluation

Primary configs:

- `configs/koopman/phi.json`
- `configs/koopman/residual.json`
